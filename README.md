# game-running-man
A clone of Google Chrome dinosaur game

**Description:** This is my first game I have ever made with java. It can be ugly, but it is fun to play.

**Features:**
* Character Select
* Random Monster Spawn System
* Unlike Dinosaur game, this game is finite and you could win.
